第一課：網站是如何運作的？

前言

當我們在瀏覽器輸入一個網址，到看見網頁畫面的這段時間，到底發生了什麼事呢？這就是我們今天要探討的主題。

基本流程

簡單來說，流程如下：

  1. 使用者在瀏覽器輸入網址 (URL)。
  2. 瀏覽器向 DNS 伺服器查詢這個網址對應的 IP 位址。
  3. 瀏覽器根據 IP 位址，向該主機（伺服器）發出請求 (Request)。
  4. 伺服器收到請求後，回傳對應的 HTML, CSS, JS 等檔案給瀏覽器。
  5. 瀏覽器接收到檔案後，將其渲染成我們看到的網頁畫面。
